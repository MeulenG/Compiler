## Getting Started

Follow the commands below on Linux to get setup:
1. Clone the repo
   ```sh
   git clone https://github.com/MeulenG/Compiler-Dependencies.git
   ```
2. Install The Dependencies
   ```sh
   chmod a+x Dev.sh
    ./Dev.sh
   ```
